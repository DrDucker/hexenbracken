About this Project

Like The Hexenbracken before it, this is another attempt to crowdsource a hex crawl. This time I turned the resulting comment thread into a Google Doc, which was then cleaned up with the help of Brendan Strejcek, James Young, Ben Hannigan and Joey Lindsey. A bunch of Python code turns that into the page you see.

A big thank you to Zak Smith for organizing and to all the contributors: {{ authors }}.
